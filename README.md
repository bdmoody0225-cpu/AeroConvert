# AeroConvert

A professional aerospace engineering toolkit written in Python.

## Features

- Engineering Unit Conversion
- Aircraft Performance Calculations
- Rocket Propulsion Tools
- Orbital Mechanics
- Standard Atmosphere Calculator
- Engineering Constants
- Material Database
- Planet Database
- Propellant Database

## Planned Features

- GUI using CustomTkinter
- Scientific plotting
- JSON engineering databases
- Numerical analysis tools
- Mission planning utilities

## Technologies

- Python
- NumPy
- SciPy
- Matplotlib
- Rich
- PyTest

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/AeroConvert.git
cd AeroConvert
pip install -r requirements.txt
python main.py
```
